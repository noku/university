### Software Product Engineering

> Product engineering refers to the process of designing and developing a device, assembly, or system such that it be produced as an item for sale through some production manufacturing process. Product engineering usually entails activity dealing with issues of cost, producibility, quality, performance, reliability, serviceability and user features.

This course is oriented more on design patterns that can be applied during creation of
software applications, also about TDD and BDD.
