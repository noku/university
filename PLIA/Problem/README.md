### Problem

Write a Prolog program which will solve the following problem. Four programmers use different editors, UNIX variations and programming languages.

- Programmers: Daisy(f), Steve(m), Liewellyn(m), Gertrude(f)
- Editors: emacs, vi, pico, joe
- UNIX: Solaris, AIX, Linux, BSD
- P. Languages: C++(OO), Java(OO), Perl, Lisp

Using following keys find which tools each programmer use:

- _Emacs_ user is a man which prefer _Lisp_.
- Woman which uses _pico_ prefers BSD, but doesn't use OO languages.
- C++ Programmer uses AIX and _joe_.
- Daisy prefers Java, but thinks that Solaris is not an good optioin.
- Liewellyn doesn't understand AIX.
